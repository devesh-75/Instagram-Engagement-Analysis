\# Instagram User Engagement Analysis (Big Data Project)



\## Introduction

This project analyzes Instagram user engagement using Big Data technologies and Machine Learning. The goal is to understand engagement patterns and predict whether a post will have high or low engagement.



\---



\## Problem Statement

To analyze large-scale Instagram data and predict user engagement using PySpark and ML techniques.



\---



\## Dataset

\- Generated synthetic dataset (1000+ rows)

\- Features include:

&#x20; - Likes, Comments, Shares

&#x20; - Followers

&#x20; - Post type (image, reel, video)

&#x20; - Watch time, Hashtags, Caption length

&#x20; - Posting hour



\---



\## Tools \& Technologies

\- Python

\- PySpark (Big Data processing)

\- Spark MLlib (Machine Learning)

\- NumPy



\---



\## Working Process

1\. Data Generation

2\. Data Loading using PySpark

3\. Data Cleaning

4\. Data Analysis (aggregation)

5\. Feature Engineering

6\. ML Preparation

7\. Model Training (Logistic Regression)

8\. Prediction \& Evaluation



\---



\## Results

\- Identified best posting time

\- Found engagement trends across post types

\- Built ML model with approximately 99% accuracy (synthetic data)



\---



\## Conclusion

This project demonstrates how Big Data tools like PySpark can be used to analyze social media engagement and build predictive models.
